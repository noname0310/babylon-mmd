# 로더

이 섹션에서는 **MMD 모델**과 **애니메이션**을 로드하는 방법에 대해 설명합니다.

**babylon-mmd**가 지원하는 **MMD 모델**과 **애니메이션 포맷**은 다음과 같습니다:

**MMD 모델**: **PMX**, **PMD**, **BPMX**

**MMD 애니메이션**: **VMD**, **VPD**, **BVMD**

각 포맷에 대한 자세한 정보는 다음 섹션을 참조하세요:

- [MMD 모델 로더](./mmd-model-loader)
- [MMD 애니메이션 로더](./mmd-animation-loader)
